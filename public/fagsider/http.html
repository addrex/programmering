HTTP

HTTP står for Hyper Text Transfer Protocol, og er reglene (protokollen) nettlesere og servere bruker for å kommunisere. Du skriver inn et nettsted, så sender nettleseren en HTTP-forespørsel til serveren som sender nettsiden tilbake. Men HTTP er ikke sikkert, og her kommer HTTPS inn.

HTTPS

Forskjellen på HTTP og HTTPS, er at HTTPS er mye sikrere. HTTP er ikke kryptert, og man kan derfor se dataene som blir sendt. Men HTTPS derimot er kryptert, som gjør at man ikke kan se dataene som blir sendt over. Derfor bruker alle nye nettsider nå HTTPS.
