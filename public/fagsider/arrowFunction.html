Arrow Functions

Arrow functions er en kortere måte å skrive funksjoner i JavaScript. I stedet for å bruke function-nøkkelordet, kan vi skrive koden mer kompakt med =>. For eksempel: const add = (a, b) => a + b; gjør akkurat det samme som function add(a, b) { return a + b; }.

Hvorfor bruke Arrow Functions?

Arrow functions gjør koden enklere å lese, spesielt når man skriver korte funksjoner. De har også en spesiell oppførsel: de binder ikke sin egen this-verdi. Det betyr at arrow functions ofte er nyttige når man jobber med metoder inne i objekter eller callback-funksjoner.

Et praktisk eksempel kan være når vi jobber med arrays. Hvis vi vil lage en ny array med tallene ganget med 2, kan vi gjøre det slik: const numbers = [1, 2, 3, 4]; const doubled = numbers.map(n => n * 2); Resultatet blir [2, 4, 6, 8].
